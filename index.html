📚 GitHub Pagesについて

GitHub Pagesは、GitHubリポジトリから直接静的なウェブサイトをホスティングできる無料のサービスです。

個人のポートフォリオ、プロジェクトのドキュメント、ブログなど、様々な用途に活用できます。

🎯 始め方

  1. リポジトリの Settings → Pages に移動
  2. Source で適切なブランチを選択
  3. 保存して数分待つ
  4. 公開されたURLにアクセス

🗺️ 旅程スケジュール作成アプリ

自分で旅程を作成して管理できるアプリです。タイムライン形式で旅の計画を作成し、時系列で予定を閲覧できます。データはブラウザに保存されるため、いつでも確認・編集が可能です。
